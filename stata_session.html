stata_session

A class for representing a Stata session

Some parts adapted from the stata_kernel version, limited for now to variables, globals, locals, scalars, matrices, and file names.

StataSession.run_as_program_w_locals

 StataSession.run_as_program_w_locals (std_code)

After break_out_prog_blocks, run noecho, inserting locals when needed

run_direct(dedent("""
    macro drop _all
    local local1 = 1
    local local2 "two"
    local local3 `""3""' 
    """), quietly=True)
test_instance.clear_suggestions()
test_instance.run_as_program_w_locals("""disp `"`local1' `local2' `local3'"' """)

1 two "3"
code = '''\
local test1 "blah blah"
local test2 "blah"
'''
test_instance.clear_suggestions()
test_instance.run_as_program_w_locals("""disp `"`local1' `local2' `local3'"' \n""" + code)
test_eq(test_instance.get_local_dict(), 
        {'test2': 'blah',
         'test1': 'blah blah',
         'local1': '1',
         'local2': 'two',
         'local3': '"3"'})
